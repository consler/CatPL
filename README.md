# CatPL
**Easy to use programming language, designed to make cross-platform graphical applications for Windows, Linux, Mac, BSD and Android.**

### Alpha features:
- [x] Basic arithmetics — +, -, *, /
- [x] Variables — assignment and usage
- [ ] Indentation check
- [ ] Events:
- -  [x] OnStart
- -  [ ] OnClick
- [ ] If statements
- [ ] While loops
- [ ] Individual scenes
- [ ] Graphical elements
- [ ] Auto float conversion
- [ ] Functions

### Example syntax:
```
onStart:
    log("yo")
    x = 12
    log(x)
    log("yoo" + 12)

onStart:
    y=1
    log(y-1)
```
