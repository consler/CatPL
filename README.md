# CatPL
**Easy to use programming language, designed to make cross-platform graphical applications for Windows, Linux, Mac, iOS and Android.**

### Alpha features:
- [x] Basic arithmetics — +, -, *, /
- [x] Variables — assignment and usage
- [x] Indentation check
- [x] Auto float conversion
- [ ] Events:
- -  [x] OnStart
- -  [ ] OnClick
- [x] If statements
- [x] While loops
- [x] Tables
- [ ] Functions
- [ ] Graphical elements
- [ ] Individual scenes

### Example syntax:
```
onStart:
    print("Hello world!")
    i=0
    table[]
    
    while(i<10):
        i=i+1
        table[i] = "i is "  + i
        
    print(table[8])
```
